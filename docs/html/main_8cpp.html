int main ( int  argc,
char *  argv[] 
)

Função principal do programa.

Esta função valida os argumentos fornecidos na linha de comando e executa as operações desejadas em uma imagem, utilizando a classe Editor. Se os argumentos forem inválidos, exibe uma mensagem de uso explicativa.

Parameters
argcNúmero de argumentos passados na linha de comando.
argvVetor de argumentos, onde:
  • argv[0] é o nome do programa.
  • argv[1] é a operação a ser realizada (como "gray", "enlarge", etc.).
  • argv[2] é o caminho para o arquivo de imagem de entrada.
Returns
Retorna 0 ao finalizar com sucesso.
